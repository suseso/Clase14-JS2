- Bucle "do..while" en JS -

El bucle do..while recorre un bloque de código siempre que una condición específica sea verdadera, pero a diferencia de while, como la condición se evalúa al final, el bloque de código siempre se ejecuta al menos una vez:

do {
// bloque de código a ejecutar
}
while (condición)

Es muy importante que esa condición en un momento pase de ser verdadera a falsa, sino se tiene un loop infinito.
